Período dos Insights
Escolha o período: Atual é o mês em andamento; Anterior é o mês passado fechado; 30D inclui hoje e os 29 dias anteriores; 6M e 12M recuam meses de calendário; Personalizado permite escolher datas; Tudo mostra todo o histórico.
Como ler os indicadores: os dados consideram a conta ativa e o período selecionado. P&L líquido = Resultado + Swap/Corretagem+IR. Capital Base = saldo inicial do período + depósitos positivos; Capital Ajustado por Fluxos = saldo inicial + depósitos + saques. Percentuais sem denominador positivo válido aparecem como N/A. O bônus é um controle paralelo: não integra capital real, saldo, P&L, performance nem a composição patrimonial.

Sobre o sistema
EP CashFlow

Painel local para acompanhar saldo, lançamentos, performance e múltiplas contas/corretoras em um único arquivo HTML.

v1.3.3 Dados locais Desktop/Laptop recomendado
O que o app faz
  • Acompanha a evolução do saldo, ganhos, perdas, depósitos, saques, upside e drawdown.
  • Organiza lançamentos consolidados por dia, semana ou mês, conforme sua rotina.
  • Mostra Dashboard e Insights por período para facilitar a leitura da performance.
Privacidade, contas e backup
  • Os dados ficam no navegador/dispositivo. O app não possui login e não envia seus lançamentos para servidor externo.
  • Permite múltiplas contas/corretoras no mesmo arquivo, cada uma com seus próprios dados e modo FOREX/B3.
  • O Backup JSON salva todas as contas e é o caminho seguro para restaurar dados ou trocar de versão.
