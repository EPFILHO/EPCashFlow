Sobre o sistema
EP CashFlow

O EP CashFlow é um painel local para acompanhar a evolução de uma conta de operações em um único arquivo HTML. Ele ajuda a registrar lançamentos, visualizar saldo, acompanhar resultados e manter backups sem depender de planilhas externas.

Resumo do app
  • Dashboard com evolução do saldo, ganhos, perdas, aportes, retiradas, upside e drawdown.
  • Lançamentos com resultado, custos/ajustes, depósito/saque e observações.
  • Insights por período para acompanhar a performance da conta com mais clareza.
  • Backup em JSON para restaurar os dados e exportação CSV para análise em planilhas.

Perguntas frequentes
Q&A

Respostas rápidas sobre uso, dados, backups, períodos, compatibilidade, termos, privacidade, licença, pagamento e principais recursos do EP CashFlow.

Preciso de internet para usar?
O EP CashFlow foi pensado para uso local: seus dados ficam no navegador e não são salvos em nuvem. A internet pode ser necessária para carregar recursos externos na primeira abertura e para acessar links como a GitHub Page do projeto.
Posso usar o EP CashFlow no celular ou tablet?
A versão atual foi pensada e recomendada para uso em Desktops/Laptops. O EP CashFlow não foi testado oficialmente em Smartphones/Tablets. Algumas funções podem até funcionar em navegadores móveis modernos, mas a experiência pode ser limitada, principalmente em tabelas, gráficos, importação/exportação de arquivos e backups.
O app coleta ou envia meus dados?
Não. O EP CashFlow não envia seus lançamentos para servidor externo. Os dados ficam no seu navegador, usando armazenamento local/IndexedDB.
Onde meus dados ficam salvos?
Eles ficam no navegador e nesta cópia do app. Se você usar outro navegador, outro computador, limpar os dados do site ou abrir uma cópia com outro nome/caminho, o histórico pode não aparecer. Para levar os dados para outra cópia, use Backup JSON e depois Importar Backup.
Posso usar várias corretoras ou contas?
Sim. O recomendado é usar uma cópia separada do HTML para cada corretora ou conta, com nomes diferentes, como CashFlow-XP.html e CashFlow-BTG.html. Em cada cópia, configure corretora, apelido e número da conta para deixar os backups bem identificados.
O EP CashFlow é um diário de trades?
Não exatamente. A proposta não é registrar cada trade individual, setup, ativo, horário de entrada ou saída. O EP CashFlow foi pensado para acompanhar a evolução da conta por lançamentos consolidados. Você pode lançar o resultado final do dia, da semana ou do mês, conforme a frequência que preferir.
Como funciona o backup?
O app salva automaticamente no navegador, mas o arquivo restaurável é o JSON. Na tela Backup, baixe o Backup JSON sempre que quiser guardar uma cópia segura. O app mostra quando existem alterações pendentes. O CSV serve mais para abrir o histórico em planilhas.
Como troco de versão sem perder meus dados?
Na versão antiga, exporte um Backup JSON. Depois, abra a nova versão e importe esse arquivo na tela Backup. Esse é o caminho mais seguro para atualizar o app mantendo o histórico.
Como funcionam os períodos dos gráficos e Insights?
Atual mostra o mês em andamento. Anterior mostra o mês passado fechado. 30D, 6M e 12M mostram janelas móveis contadas para trás a partir de hoje. Tudo mostra todo o histórico salvo. Personalizado permite escolher uma data inicial e uma data final. O período escolhido é sincronizado entre o Dashboard e os Insights.
Qual é a diferença entre FOREX e B3?
No modo FOREX, o campo de custo/ajuste aparece como Swap e o app permite bônus. No modo B3, o campo aparece como Corretagem + IR e os campos de bônus ficam ocultos.
O que são “Com saque” e “Sem saque”?
Com saque mostra o saldo real, considerando retiradas. Sem saque mostra uma simulação de como o saldo teria evoluído se os saques não tivessem sido retirados da conta.
Posso editar ou apagar lançamentos?
Sim. Na tela Lançamentos, use as ações do histórico para editar ou excluir registros. Depois de qualquer alteração, o app marca que há backup pendente para você baixar um JSON atualizado.
